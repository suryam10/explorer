Github Command and Its Meanings

github commands

  1. git clone git@github.com:jeevaengg21/web_development.git - to clone a code repository into local machine or computer
  2. ssh-keygen-t ed25519 -C "your@gmail.com" - to generate ssh key
  3. git status - it is used to check show the working status
  4. git add . - it is used to add all the file content without specifing filename
  5. git commit -am"initial commit" - it is used to describe what we changed in that file
  6. git push - it is used to push the code to github repository
  7. git config --global user.email"your@email.com"-it is used to identify the user
  8. git config --global user.name"your_name" - it is used to identify the user name
