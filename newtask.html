Git Commands

How to clone repository?

git clone - it is used to clone the repository in local machine

How to check repository status?

git status - this command is used to check the status of the local files

How to generate ssh key?

ssh-keygen-t ed25519 -C "your@gmail.com" - this is used to generate ssh key

How to add the files?

git add . - this is used to add many files without specifing

How to record changes?

git commit -am"initial comment" - this is used to record changes in the code repository

How to update the code from local machine?

git push - this is used to update the code or upload to code repository

How to identify the user?

git config --global user.email"your@email.com - it is used to identify the user

How to identify the username?

git config --global user.name"your_name" - this used to identify the username

How to publish the repository in Github?

  1. got to the settings
  2. select the pages menu
  3. open the drop down menu under branch
  4. select the branch as main and click save

Now the website is published!
